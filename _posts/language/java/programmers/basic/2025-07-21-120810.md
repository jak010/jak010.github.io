# 문제 설명

정수 `num1`, `num2`가 매개변수로 주어질 때, `num1`를 `num2`로 나눈 나머지를 return 하도록 solution 함수를 완성해주세요.

# Solution

```java
class Solution {
    public int solution(int num1, int num2) {
        return num1 % num2;
        
    }
}
```
