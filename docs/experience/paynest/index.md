### 기여 및 성과

| 프로젝트 / 과제 | 상세 내역 |
| :--- | :--- |
| **중단된 가상계좌 결제 서비스 분석 및 재개 검증 환경 구축** | • **문제**: 가맹점의 자금세탁 악용 이슈로 가상계좌 결제 서비스 운영이 중단되어, 운영망에서의 정상적인 결제 흐름과 내부 시스템 동작 상태를 직접 확인하기 어려운 상황 발생. 신규 사업 재개를 위해 기존 연동 규격 검증 및 금융망 연동 인프라 재구축이 필요함.<br><br>• **해결**: 레거시 API 명세와 소스 코드를 전수 분석하여 가상계좌 발급부터 고객 입금, 입금 확인 통지, 결제 완료, 정산 데이터 생성에 이르는 전체 데이터 라이프사이클을 매핑함. 외부 핀테크 API 연동 절차를 확인하고, 테스트 은행(경남은행) 연동망 설정을 통해 격리된 테스트 가상계좌 결제 환경을 확보함. 연동 검증용 스크립트를 작성하여 API 오류 대응 및 예외 흐름(타임아웃, 중복 통지 등)을 검증함.<br><br>• **성과**: 가상계좌 서비스의 전 주기 검증 환경을 안전하게 복원하고 시스템 불일치 지점을 규명하여 서비스 재개 가능 상태를 검증함. 예외 케이스 검증 체계를 확보하여 향후 가맹점 연동 시 발생할 수 있는 장애 리스크를 최소화함. |
| **가상계좌 서비스 재개를 위한 시스템 검증 및 운영 문서화** | • **문제**: 기존 서비스 문서의 현행화가 되어 있지 않아 실제 구동 중인 시스템 아키텍처 및 세부 비즈니스 규칙(상태값 변화, 정산 대사 매칭 등)의 정확한 일치 여부 파악이 곤란함.<br><br>• **해결**: API 동작 및 트랜잭션 상태 변화를 실시간으로 테스트 검증하고, 가상계좌 발급, 결제 상태 천이, 정산 배치 동작 등 시나리오별 검증 결과를 토대로 시스템 현황 분석 문서를 작성함.<br><br>• **성과**: 서비스 재개에 앞서 세부 기능 규격서와 운영 가이드를 현행화하여 내부 기술 부채를 해결하고, 신규 사업 추진을 위한 의사결정 및 운영 프로세스의 명확한 기준 문서로 활용함. |
