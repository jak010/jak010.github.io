## 주요 정보통신기반시설 인프라 보안 취약점 진단

### 프로젝트 개요

| 항목        | 내용                                                             |
| --------- | -------------------------------------------------------------- |
| **기간**    | 2020.09 ~ 2020.12                                              |
| **업무**    | 공공기관 정보보호 가이드라인을 기준으로 서버 환경의 보안 취약점을 분석 및 보안 강화를 위한 개선 방안을 제시 |
| **대상 환경** | Linux (RedHat, CentOS), Windows Server, UNIX                         |

### 기여 및 성과

| 역할 / 수행 | 기여 및 성과 |
| :--- | :--- |
| **보안 취약점 진단 수행** | 공공기관을 대상으로 주요정보통신기반시설 기술적 취약점 분석·평가 가이드 문서 기반으로 다수의 보안 점검 항목을 활용하여 Linux 및 Windows 서버의 보안 취약점 진단 수행 |
| **보안 설정 점검 및 취약점 도출** | 시스템 계정 및 패스워드 정책, 관리자 계정 관리, Telnet·FTP 비활성화, SSH 보안 설정 등 운영체제 및 서비스 보안 설정 전반을 점검하고 취약 항목 도출 |
| **보고서 작성 및 개선 방안 제시** | 취약점 분석 결과 보고서 작성에 참여하여 가이드라인 준수 현황을 정리하고, 보완이 필요한 항목에 대한 개선 방안 및 조치 권고 제시 |

## 헬스케어 빅데이터 수집 및 전처리 자동화 스크립트 구축

### 프로젝트 개요


| 구분        | 내용                                                   |
| --------- | ---------------------------------------------------- |
| **기간**    | 2020.09 ~ 2020.12                                    |
| **업무**    | 헬스케어 데이터 수집을 위한 인스타그램 인플루언서 게시글 및 댓글 데이터 수집 웹 크롤러 개발 |
| **기술 스택** | Python, BeautifulSoup, Selenium, Google API                      |


### 기여 및 성과

| 역할 / 수행 | 기여 및 성과 |
| :--- | :--- |
| **웹 크롤러 개발** | Python, Selenium, BeautifulSoup를 활용하여 인스타그램 게시글 및 댓글 데이터를 수집하는 웹 크롤러 개발 |
| **동적 데이터 수집 구현** | 동적 웹 페이지의 로그인, 스크롤 및 HTML 파싱을 구현하여 게시글·댓글 데이터를 안정적으로 수집 |
| **데이터 전처리 및 데이터셋 구축** | 수집 데이터를 전처리 및 구조화하여 헬스케어 빅데이터 분석에 활용 가능한 데이터셋 구축 |
