# Skills

## Technical Competency

4년 이상의 백엔드 개발 경험을 바탕으로 **Python 기반 백엔드 서비스 설계부터 데이터 처리, 운영 환경 개선까지 전반적인 시스템을 구축**해 왔습니다. 단순히 특정 프레임워크를 사용하는 것보다 **서비스 규모와 요구사항에 맞는 기술을 선택하고, 안정적으로 운영 가능한 구조를 설계하는 것**을 강점으로 가지고 있습니다.

[FastAPI]({% link docs/side-projects/podpod/index.md %}), [Django]({% link docs/experience/emforce/index.md %}), [Spring Boot]({% link docs/experience/paynest/index.md %})를 활용하여 REST API와 비즈니스 로직을 설계하고, [SQLAlchemy]({% link docs/side-projects/poomasi/index.md %}), [JPA]({% link docs/experience/paynest/index.md %}), [MyBatis]({% link docs/experience/paynest/index.md %}) 등 다양한 데이터 접근 기술을 프로젝트 특성에 맞게 적용해 왔습니다. 또한 [WebSocket 기반 실시간 통신 서버]({% link docs/experience/basetrade/index.md %})와 비동기 처리 환경을 구축하며 [Daphne]({% link docs/experience/basetrade/index.md %}), **aiohttp**, [Redis]({% link docs/experience/basetrade/index.md %})를 활용한 성능 최적화와 안정적인 서비스 운영 경험을 보유하고 있습니다.

데이터 처리 분야에서는 [BeautifulSoup]({% link docs/experience/soulsoft/index.md %}), [Selenium]({% link docs/experience/emforce/index.md %}), **cloudscraper**, [HTTP API 수집 리팩토링]({% link docs/experience/emforce/index.md %}) 등을 활용한 대규모 데이터 수집 경험과 함께, [Gemini API]({% link docs/side-projects/poomasi/index.md %})와 **LangChain**을 활용한 LLM 기반 데이터 가공 및 자동화 시스템을 구축해 왔습니다. 서로 다른 형태의 데이터를 하나의 구조로 통합하고, AI를 활용하여 서비스에서 활용 가능한 데이터로 변환하는 데이터 파이프라인 설계를 강점으로 가지고 있습니다.

운영 환경에서는 [Docker]({% link docs/experience/basetrade/index.md %}) 기반 컨테이너 환경과 [AWS]({% link docs/experience/emforce/index.md %})(EC2, ECS, Lambda, RDS, S3, CloudWatch)를 활용하여 서비스를 구축하고 운영했으며, [GitHub Actions]({% link docs/side-projects/poomasi/index.md %}), [GitLab CI/CD]({% link docs/experience/basetrade/index.md %}), [Jenkins]({% link docs/experience/paynest/index.md %})를 활용한 배포 자동화 경험을 보유하고 있습니다. 또한 [Nginx]({% link docs/experience/basetrade/index.md %}), [Infisical]({% link docs/side-projects/podpod/index.md %}), [GlitchTip]({% link docs/side-projects/podpod/index.md %}), [Sentry]({% link docs/experience/emforce/index.md %}) 등을 활용하여 보안, 모니터링, 장애 대응 체계를 구축하며 서비스의 안정성과 운영 효율성을 지속적으로 개선해 왔습니다.

새로운 기술을 빠르게 학습하는 것보다 **기술을 문제 해결의 도구로 활용하는 것**을 중요하게 생각합니다. 서비스 규모와 운영 환경을 고려하여 적합한 기술을 선택하고, 성능과 유지보수성을 함께 고려한 백엔드 시스템을 설계하는 것을 가장 큰 기술적 강점으로 가지고 있습니다.
